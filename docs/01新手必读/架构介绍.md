# 架构介绍
架构图先欠着吧，一点点东西画啥架构图

## 项目目录结构
```
.
├─com
│  └─zeta                        # 业务包，专注于业务curd
│      │  KtZetaApplication.kt   # 启动类
│      ├─common                  # 业务包通用的东西
│      │  ├─config
│      │  └─constants
│      ├─msg                     # 消息管理
│      └─system                  # 系统管理
│
└─org                            # 框架核心包
    └─zetaframework
        ├─base                   # 基础类型，包含BaseController、BaseEntity、BaseParam、BaseResult，虽然他们都不叫这个名
        │  ├─controller
        │  ├─entity
        │  ├─param
        │  └─result
        ├─core                   # 各种配置
        │  ├─constants
        │  ├─enums
        │  ├─exception           # 全局异常处理
        │  ├─jackson             # jackson配置
        │  ├─log                 # 操作日志、登录日志相关
        │  ├─mybatisplus         # mybatis plus配置
        │  ├─redis               # redis配置、redis工具类、接口限流
        │  ├─saToken             # 权限认证配置 shiro和spring security的替代
        │  ├─swagger             # swagger文档，有的人喜欢，有的人不喜欢
        │  ├─utils
        │  └─validation          # 参数校验
        └─extra                  # 扩展，一些不在我的计划中，但是又挖了坑要实现的玩意
            ├─file               # OSS文件管理
            └─websocket          # websocket
```
